# Escape-From-Dema

Escape From Dema

Escape From Dema é um minigame narrativo inspirado no universo conceitual da banda Twenty One Pilots. O projeto combina elementos de RPG e Quiz interativo para criar uma experiência de tomada de decisão onde cada escolha molda o destino do protagonista.

Sobre o Projeto

O objetivo principal é guiar um cidadão em sua jornada de fuga da cidade de Dema. Através de uma série de dilemas e desafios, o jogador percorre caminhos ramificados que levam a três finais distintos, incentivando a rejogabilidade e a exploração de diferentes estratégias.

Funcionalidades

Narrativa Dinâmica: Quiz interativo baseado em lógica de RPG que altera o final da história do personagem.

Análise de Desempenho: Ao final da jornada, o sistema apresenta uma Dashboard com métricas sobre o percurso do jogador.

Feedback Personalizado: Uma análise detalhada das escolhas feitas, permitindo que o usuário entenda seu perfil de decisão e comportamento dentro do jogo.
